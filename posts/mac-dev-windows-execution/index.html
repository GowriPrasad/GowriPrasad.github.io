Mac as Dev Device, Windows as Execution Environment

A clean, repeatable setup to use Mac for development and Windows as the execution environment using SSH and VS Code.

Step 1: Install OpenSSH Server on Windows#

On Windows (target machine):

  1. Open Settings
  2. Search for Optional features
  3. Click View features
  4. Search for OpenSSH Server
  5. Select it and Install

This enables Windows to accept SSH connections.

Step 2: Verify OpenSSH Installation and Service#

Open PowerShell as Administrator and run:

Get-WindowsCapability -Online | Where-Object Name -like 'OpenSSH*'

Ensure both show State : Installed.

Start and enable the service:

Start-Service sshd
Set-Service sshd -StartupType Automatic

Step 3: Allow SSH Through Windows Firewall#

Get-NetFirewallRule -Name *ssh*

If missing:

New-NetFirewallRule -Name sshd -DisplayName "OpenSSH Server" -Enabled True -Direction Inbound -Protocol TCP -Action Allow -LocalPort 22

Step 4: Generate SSH Key on Mac#

ls ~/.ssh/id_ed25519.pub

If missing:

ssh-keygen -t ed25519

Step 5: Copy Public Key on Mac#

cat ~/.ssh/id_ed25519.pub

Copy the output.

Step 6: Add Public Key on Windows (Administrator User)#

File location (mandatory):

C:\ProgramData\ssh\administrators_authorized_keys

Create and edit:

New-Item -Force -Path "$env:ProgramData\ssh\administrators_authorized_keys" -ItemType File
notepad "$env:ProgramData\ssh\administrators_authorized_keys"

Paste the key and save.

Step 7: Apply Strict Permissions (Required)#

$path = "$env:ProgramData\\ssh\\administrators_authorized_keys"
icacls $path /inheritance:r
icacls $path /grant "Administrators:F"
icacls $path /grant "SYSTEM:F"

Step 8: Test SSH from Mac#

ssh <windows-username>@<windows-ip>

Step 9: Add Key to macOS Keychain#

ssh-add --apple-use-keychain ~/.ssh/id_ed25519

Step 10: Configure SSH on Mac#

Edit config:

nano ~/.ssh/config

Add:

Host *
  AddKeysToAgent yes
  UseKeychain yes
  IdentityFile ~/.ssh/id_ed25519

Step 11: Create SSH Alias#

Host win
  HostName <WINDOWS_IP_ADDRESS>
  User <WINDOWS_USERNAME>

Connect using:

ssh win

Step 12: VS Code Remote Explorer#

  1. Open VS Code on Mac
  2. Install Remote Explorer
  3. Open Remote Explorer ‚Üí SSH
  4. Connect to win

Final Thought#
